## Overview
This project is a custom-built version of the **Vecta Corp** website created entirely using **Bootstrap** components and utilities.  
It demonstrates the use of responsive layouts, navigation bars, carousels, tabs, collapse, and scrollspy to design a clean and professional website interface.

---

## Features Implemented

### 1. **Bootstrap Integration**
- All required Bootstrap CSS and JS files are correctly included.  
- Project files are organized into proper folders (`css/`, `js/`, `images/`).  

### 2. **Main Page Layout – `index.html`**
- Includes a **navigation bar**, **header**, **main content area**, and **footer**.  
- Content area divided into **3 equal Bootstrap columns** using the grid system.  

### 3. **Navigation Menu**
- A **floating navigation bar** (not fixed/static).  
- Menu items: `Home`, `About Us`, `Solutions`, `Support`, `Contact Us`.  
- `About Us` dropdown: Overview, Company History, Management, Careers.  
- `Solutions` dropdown: vProspect, vConvert, vRetain.  

### 4. **Carousel**
- Full-width **Bootstrap carousel** below the navigation bar.  
- Includes rotating software-themed images with captions and “More Info” buttons.  

### 5. **Tabs Component**
- The **center column** includes 3 product descriptions using Bootstrap **Tabs**.  
- Left column: welcome text.  
- Footer: includes horizontal rule and footer note.  

### 6. **Collapse Component – Testimonials**
- Right column includes **2 client testimonials** using Bootstrap **Collapse**.  
- Clicking each client’s name reveals their testimonial.  

### 7. **Solutions Page**
- `solutions.html` includes a **right-side navigation menu** for vProspect, vConvert, and vRetain.  
- Uses **Scrollspy** to highlight active sections as users scroll.  
- Each section includes provided content from the Assets folder.
